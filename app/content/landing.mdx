# Take Two!

Take two is a Hack Club YSWS to get you building your hardware projects. Get yourself a second take. Here's how it works:

1. Try to build your project! Write a BLOG.md file on your experiences!
2. If you need to make changes, get up to $75 to get new parts!
3. If it did work, get verified hardware hacker status. This will be used for later :eyes:
